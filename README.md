# code-quiz

A simple three question timed quiz about Javascript basics. When the user presses the "Start" button, the timer and a multiple choice quiz begins. Answering correctly earns you a point, answering incorrectly docks you a point and costs 2 seconds against the timer. When you finish answering the questions or when the timer reaches 0, your score is shown.
